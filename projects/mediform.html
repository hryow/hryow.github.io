the project

+ github + devpost

Mediform is a website that is designed to store users' critical medical information for ease of access. The project was initially supposed to be an application for first-responders that would read NFC cards containing the information. However, this proved to be too ambitious so we shifted our approach to creating a website instead.

The goal of this project was to improve healthcare by allowing first-responders to provide medical aid tailored to individual needs. However, we were unable to fully realize our original goal due to our collective lack of experience in programming as first-years, and have created a website that reads information from medical files.
